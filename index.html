이용방법

마블 룰렛 사용법

1. 참가자 입력

왼쪽 텍스트 영역에 참가자 이름을 입력합니다.

  • 쉼표(,) 또는 줄바꿈으로 구분
  • 이름*숫자: 같은 이름의 마블 여러 개 (예: 철수*3)
  • 이름/숫자: 마블 크기 조절 (예: 영희/2)

2. 옵션 설정

  • 맵: 다양한 맵 테마 선택
  • 녹화: 결과를 자동으로 녹화
  • 당첨 기준: 1등 또는 꼴등을 당첨자로 설정
  • 스킬 활성화: 마블이 주변을 밀어내는 스킬 사용

3. 시작하기

시작 버튼을 눌러 추첨을 시작합니다. 마블들이 떨어지며 당첨자가 결정됩니다!
